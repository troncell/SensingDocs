# 1.派样机介绍
接入不同厂家的贩卖机设备，连通SensingStoreCloud平台，获取货道信息、活动内容、货道库存等信息数据。在贩卖机设备上安装Sample应用，展示活动内容，用户可参与活动，领取贩卖机上的样品。补货人员可定期对派样机进行补货。

使用环境：派样机设备目前仅支持Android

## 1.江森测试小车项目介绍

简介：更智能化的检测手段，自动读取等技术，引入辅助 PDA 设备， 可移动式办公(后续拓展)。兼容多种电气测试仪，检测的过程可记录，可追溯。测试的项目同步于 MES，必要的数据回传到 MES 系统

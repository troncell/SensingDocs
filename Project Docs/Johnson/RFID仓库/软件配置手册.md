# 1.上位机

当电脑系统出现故障或由于其他原因导致必须更换电脑时，江森的IT人员可依照本手册在新的电脑上重新部署设备运行必须的系统环境。

此处的环境均在win11上验证成功，请准备好win11系统。

另外，装完环境要正确运行上位机软件，是需要连上服务器的，所以准备好的PC系统要和服务器在一个网络内。（验证方法就是用此台电脑直接访问服务器的站点，可以打开后台web管理页面，站点地址为：[http://192.168.3.105:8070/#/login/8307](http://192.168.3.105:8070/#/login/8307)）

1.1环境安装

1.安装打印机驱动x86,软件包地址：\\Troncell_NAS920\实施交付\2026年项目交付备份\2026-3 江苏RFID库房\sdk\打印机

2.安装net10.0和.net core，官方下载地址：[下载 .NET 10.0 (Linux、macOS 和 Windows) | .NET](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0)

1.2配置文件

（1）RfidTagPrinting.App.dll.config

```

<?xml version="1.0" encoding="utf-8" ?>

<configuration>

    `<appSettings>`

    `<add key="MachineConfigFile" value="MachineConfigs-001.xml" />`  //启用得设备配置文件

    `<add key="IsLocalRun" value="False"/>`  //是否本地模式   (目前未开发完成)

    `</appSettings>`

</configuration>
```

（2）MachineConfigs-001.xml

主要修改：打印模板得文件：Config="rfid-printer.xml

后台IP地址：IpAddress="http://47.117.98.11:8081 租户id可配：TenantId="4"

是否模拟：IsMock="false"，可以模拟打印，不启用打印机

```
<MachineConfig>
<Devices>
<Printers>
<Printer Id="Rfid-Printer" Name="RfidPrinter" Type="rfid" Protocol="Com" IpAddress="192.168.1.21" Config="rfid-printer.xml" IsMock="false"/>
</Printers>
</Devices>
<Services>
<Service Id="001" Name="BacKEndWebApi" IpAddress="http://47.117.98.11:8081" Port="" Protocol="http" Comment="创思的服务后台" TenantId="4" IsMock="False"/>
</Services>
</MachineConfig>
```

（3）rfid-printer.xml
主要修改：启用得打印模板得文件目录和文件

```
<Configs>
<PrintTemplate Path="./Print/Templates/HasMaterialName.dlt"> </PrintTemplate>
</Configs>
```

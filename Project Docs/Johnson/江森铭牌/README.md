# 1.江森铭牌项目介绍

1.简介：全自动金属铭牌柔性打印系统(可兼容手动):，最深刻度为 0.1mm，铭牌尺寸 1.支持激光打印，种类多大几十种。支持纸质打印，纸质打印机设备由客户提供 。柔性自动上下料机构。铭牌内容信息来源多样，MES 系统，设计图纸等，需要自动获取信息，特殊情况可人工干预。铭牌控制系统，把获取的铭牌信息，根据型号，可控制机械部分进行打印，同时提供友好的人工操作界面。

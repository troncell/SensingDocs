# 小程序打包上传发布

> 适用场景：将小程序代码通过微信开发者工具上传，经微信开放平台添加到模板库，再通过服务器脚本发布体验版或正式版。

## 前置准备

- 已安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
- 已在微信开放平台绑定第三方平台及目标小程序。
- 已配置服务器连接工具（Xshell / FileZilla）并拿到 `139.224.15.28` 访问权限。
- 已拿到本次发布所需：小程序 `APPID`、开放平台账号、服务器目录 `/mnt/wxplatform` 权限。

> ⚠️ 本文档包含账号密码等敏感信息，请勿外传，建议后续迁移至密钥管理工具或环境变量中管理。

## 1. 查看小程序信息

### 1.1 查看 APPID

在微信开发者工具中打开项目，点击右上角【详情】，即可看到小程序的 **APPID**。

![1784254338181](image/小程序打包发布/1784254338181.png)

### 1.2 查看项目配置（域名信息）

在【详情】→【项目配置】中查看域名信息。该域名需要在小程序后台提前配置好，**绑定第三方平台前必须先配置**，否则需要先解绑第三方平台才能看到并配置。

![1784254391812](image/小程序打包发布/1784254391812.png)

### 4.3 发布体验版

#### 步骤 1：更新 Token

进入项目目录并执行 `read_token.js`：

```bash
cd /mnt/wxplatform/troncell
node read_token.js
```

![1784256586619](image/小程序打包发布/1784256586619.png)

#### 步骤 2：启用体验版相关代码

在 `audit.js` 中确保以下代码处于启用状态：

```js
await commit();
await get_qrcode();
```

![1784256843149](image/小程序打包发布/1784256843149.png)

#### 步骤 3：执行脚本并下载二维码

运行 `audit.js`，同级目录会生成体验版二维码图片，下载后用微信扫码即可查看。

```bash
node audit.js
```

![1784256825590](image/小程序打包发布/1784256825590.png)

![1784256878532](image/小程序打包发布/1784256878532.png)

#### 步骤 4：扫码预览

使用微信扫描体验版二维码即可进入小程序预览。

![1784257118766](image/小程序打包发布/1784257118766.png)

### 4.4 发布正式版

#### 步骤 1：上传媒体资源

```bash
node uploadmedia.js
```

![1784257405387](image/小程序打包发布/1784257405387.png)

#### 步骤 2：提交审核

在 `audit.js` 中：

1. 注释发布体验版的代码（`commit()` 和 `get_qrcode()`）。
2. 启用 `submit_audit()`。

> 必须先提交审核，审核通过后才能正式发布。

![1784257218907](image/小程序打包发布/1784257218907.png)

#### 步骤 3：执行审核脚本

```bash
node audit.js
```

![1784257474859](image/小程序打包发布/1784257474859.png)

#### 步骤 4：查看审核状态

在 `audit.js` 中启用：

```js
get_latest_auditstatus();
```

然后执行脚本查看审核结果。

![1784257501594](image/小程序打包发布/1784257501594.png)

![1784257557863](image/小程序打包发布/1784257557863.png)

#### 步骤 5：正式发布

审核通过后，在 `audit.js` 中启用发布相关代码并再次执行：

```bash
node audit.js
```

![1784257994287](image/小程序打包发布/1784257994287.png)

## 常见问题

| 问题                     | 可能原因                         | 处理方式                                         |
| ------------------------ | -------------------------------- | ------------------------------------------------ |
| 上传后草稿箱看不到版本   | 第三方平台未绑定该小程序         | 先完成[3.3 绑定开发小程序](#33-绑定开发小程序)   |
| 提示域名未配置           | 小程序后台未提前配置域名         | 解绑第三方平台，配置域名后再重新绑定             |
| `read_token.js` 执行失败 | Token 过期或网络异常             | 检查开放平台 Token 状态，重试或联系管理员        |
| 正式版无法发布           | 审核未通过或未查询到最新审核状态 | 先执行 `get_latest_auditstatus()` 确认审核已通过 |

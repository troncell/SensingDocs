# 自助收银

人脸支付在零售场景有着广泛的应用，在零售门店可以在收银台引进人脸支付与门店消费系统结合应用，让顾客在结算时有多种支付方式选择，提升购物体验。也可以在门店中引入人脸识别自助收银机，让顾客选购商品，自助扫描商品结算，减少排队时间。甚至还有无感支付技术。利用AI人脸识别技术，消费者出店时闸机自动识别用户ID并开门放行，同步系统提交购物车清单完成扣款。

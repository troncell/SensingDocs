# 播控介绍

采用SensingHub，SensingHub是一个本地化的多功能的Web服务，主要负责同一场所下的所有智能设备的基础设置管理、设备下的多媒体资源管理、设备中应用软件中实时事件的分发以及对应的数据收集.

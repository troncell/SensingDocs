# 人脸闸机系统

无人店能够无感识别会员身份。用户ID与进闸机时的面部识别信息绑定，成为一个连贯的行走的ID，这是无人便利店为用户提供个性化服务的前提。同时还要签署一个“免密付款”的协议，开启“即拿即走”之旅。

# 魔镜
互动魔镜介绍:互动魔镜是一种新型的互动展示系统,魔幻镜面互动系统是以背投方式的一种多媒体互动形式。镜面是一块高性能的双面显示屏幕,它具有视觉平方所有产品在清晰显像、耐光亮等优点。同时,它又是一面镜子。

# 客流
视频客流分析能够精确获取每日来到客流人次，货架关注度情况等大数据。在零售门店向智慧化改造过程中，需要实现客群分析、热区统计等需求，在这样模式下，要通过人脸识别系统获取顾客的人脸信息，进而分析顾客的性别、年龄等熟悉，及对到店时间等非识别信息记录，为商家提供客群统计分析。

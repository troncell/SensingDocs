

# SAT超声波扫描分析
非接触式超声波无损检测。在不损坏工件或原材料工作状态的前提下，对被检验部件的表面和内部质量进行检查。

无损检测：非接触、高精度、无损伤的超声检测，利用超声波对金属构件内部缺陷进行检查。

1：适用范围广，适用于金属非金属，复合材料
2：灵敏度高，实时成像
3：分层扫描，多层扫描功能
4：缺陷定位较准确
5：对面积型缺陷敏感
6：生成缺陷报告
7：对人体，工件，环境无损害

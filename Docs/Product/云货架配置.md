#### 2.2.2 **System.xml**

系统级全局配置，控制窗口分辨率、显示范围、搜索模式及调试模式。

```xml
<?xml version="1.0" encoding="utf-8"?>
<System>
  <!-- 设计分辨率：按 3840x2160 设计，在 1920x1080 等屏幕上 AutoScale 缩放到全屏 -->
  <Resolution Width="3840" Height="2160" />
  <!-- 窗体的实际显示范围，包括起始位置和大小 -->
  <DisplayBound Left="0" Top="0" Width="3840" Height="2160" AutoScale="True" />
  <!-- 搜索模式 True:正常搜索不弹出搜索面板 False:弹出搜索面板 -->
  <SearchMode Value="False">False</SearchMode>
  <!-- 是否为 Debug 模式，True 支持鼠标操作并显示调试信息 -->
  <IsDebugMode Value="True">True</IsDebugMode>
</System>
```

**System.xml 参数说明**

| 参数 | 说明 | 典型值/备注 |
|---|---|---|
| `Resolution` | 设计分辨率。整个应用按此分辨率进行布局，实际运行时可根据屏幕自动缩放。 | `3840×2160`（4K 横屏） |
| `Resolution@Width` | 设计分辨率宽度（像素）。 | `3840` |
| `Resolution@Height` | 设计分辨率高度（像素）。 | `2160` |
| `DisplayBound` | 窗体实际显示范围。多屏拼接时可通过 `Left`/`Top` 控制显示区域，通过 `Width`/`Height` 控制总显示大小。 | 通常与 `Resolution` 保持一致 |
| `DisplayBound@Left` | 显示区域左上角 X 坐标。 | `0` |
| `DisplayBound@Top` | 显示区域左上角 Y 坐标。 | `0` |
| `DisplayBound@Width` | 显示区域宽度（像素）。四连屏等场景可设为 `4320` 等。 | `3840` |
| `DisplayBound@Height` | 显示区域高度（像素）。 | `2160` |
| `DisplayBound@AutoScale` | 是否根据实际屏幕自动缩放。`True` 时自动适配，`False` 时按设计分辨率 1:1 显示。 | `True` |
| `SearchMode` | 搜索模式。`True` 为正常搜索不弹出搜索面板；`False` 为点击搜索后弹出搜索面板。 | `False` |
| `IsDebugMode` | 是否为 Debug 模式。`True` 时支持鼠标操作、显示调试信息，便于开发调试；生产环境建议设为 `False`。 | `True`（开发）/ `False`（生产） |

##### WallPage.xml 参数说明

| 参数/节点 | 说明 |
|---|---|
| `Width` / `Height` | 页面设计分辨率。当前配置为 `3840×2160`，表示按 4K 横屏设计。 |
| `Top` / `Left` | 页面在屏幕坐标系中的起始位置，通常保持 `0`。 |
| `OffsetY` | 商品瀑布流整体向下偏移的像素值，用于给顶部标题栏、分类栏等留出空间。 |
| `Rows/Row@Speed` | 瀑布流每一行的横向滚动速度。负值表示向左滚动，数值越大滚动越快。 |
| `Layout` | 布局类型，见下方详细说明。可选 `WaterFlow`（瀑布流）或 `FiveGrid`（五宫格）。 |

**布局类型说明**

- **`WaterFlow`（瀑布流）**：商品按原始图片比例横向排列，每行高度固定，系统每次把新商品放到“最短行”。整屏商品持续从右向左滚动，适合展示大量不同尺寸的商品图。
- **`FiveGrid`（五宫格）**：页面由“大图单元格”和“四宫格单元格”交替组成。每个大图单元格占一个固定大高度；四宫格单元格内部再细分为 2×2 共 4 张小图。整体仍以行横向滚动，但视觉节奏更适合突出部分重点商品。

**单元格配置节点**

| 节点 | 适用场景 | 说明 |
|---|---|---|
| `WallItem` | 普通商品 | 配置普通商品卡片（背景图、商品图、名称、价格、年龄段等）。 |
| `BigWallItem` | 大商品 / 突出商品 | 配置大卡片样式，在 `WaterFlow` 中用于 `IsBig` 商品，在 `FiveGrid` 中用于大图单元格。 |
| `WallWaterItem` | 水滴详情浮层 | 用户点击/触摸商品后，商品被“水滴”放大吸附，进入激活态，此时使用该配置。包含大图轮播、喜欢按钮、详情按钮、价格、二维码等。 |

每个单元格内部通用子节点含义：

| 子节点 | 说明 |
|---|---|
| `Background` | 单元格背景图。`Enable` 控制是否显示；`UriKind` + `FileName` 指定图片路径。 |
| `ImageRect` | 商品主图显示区域。`Left/Top/Width/Height` 为相对单元格左上角的坐标；`Enable` 控制是否显示。 |
| `LikeRect` | 喜欢/收藏按钮区域。在水滴态（`WallWaterItem`）中通常用于触发收藏。 |
| `InfoRect` | 详情按钮区域。在水滴态中点击可打开商品详情页。 |
| `NameText` | 商品名称文本。支持 `FontSize`、`FontFamily`、`Foreground`、`MaxWidth`、`Enable`、`FontWeight`。 |
| `PriceText` | 商品价格文本。参数同 `NameText`。 |
| `PromPriceText` | 促销价格文本（划线价场景），仅在 `WallWaterItem` 等水滴态配置中使用。 |
| `AgeScopeText` | 年龄段文本，例如 `3-6岁`。 |
| `QrcodeRect` | 二维码显示区域（旧版水滴态使用，新版已移除）。 |

**固定 UI 控件（`Canvas/Controls`）**

**DetailsPage.xml 参数说明**

**页面根属性**

| 参数 | 说明 |
|---|---|
| `Width` / `Height` | 详情页设计分辨率。 |
| `Top` / `Left` | 详情页在屏幕坐标系中的起始位置。 |

**顶层节点**

| 节点 | 说明 |
|---|---|
| `ActivatorInfo` | 详情页使用的自定义控件。`AssemblyFile` 为程序集文件名，`TypeName` 为完整类型名。 |
| `ShowItem` | `True` 默认选中“资源/商品图”，`False` 默认选中 SKU 信息。 |
| `LocalQrcodeUrl` | 当商品没有线上二维码时，使用的默认二维码地址。 |
| `Canvas` | 详情页根容器，内部通过 `Controls` 放置所有 UI 元素。 |

**通用控件属性**

| 属性 | 说明 |
|---|---|
| `Left` / `Top` | 控件左上角坐标，相对于父容器。 |
| `Width` / `Height` | 控件宽高。 |
| `Binding` | 数据绑定字段，如 `Title`、`Price`、`Quantity`。 |
| `IsShowBinding` | 绑定布尔字段控制控件显示/隐藏，如 `IsCommentsEmpty`。 |
| `FontSize` / `FontFamily` / `Foreground` / `FontWeight` | 文本样式。 |
| `TextWrapping` / `TextAlignment` / `Padding` | 文本布局。 |
| `Zindex` | 控件层级，数值越大越在上层。 |
| `Command` | 按钮命令，如 `CloseCommand` 关闭详情页。 |

**常用控件**

| 控件 | 说明 |
|---|---|
| `Image` | 图片。通过 `FileSource` 指定 `UriKind`（`Application` 为相对程序根目录）和 `Path` 路径。 |
| `TextBlock` | 文本块。支持数据绑定、样式和自动换行。 |
| `QrcodeControl` | 二维码控件。`QrcodeUrlBinding` 绑定二维码 URL 字段。 |
| `FlipView` | 图片轮播控件。`ItemsSourceBinding` 绑定媒体列表，`SelectedIndexBinding` 绑定当前选中索引。 |
| `MediaList` | 缩略图/资源图列表。`ItemWidth` 为单项宽度，`SelectedBrush` 为选中色，`DataContextBinding` 绑定数据源。 |
| `CustomElement` | 自定义 WPF 控件，通过 `ActivatorInfo` 加载外部程序集中的用户控件。 |
| `ImageButton` | 图片按钮。可配置 `Command` 和 `FileSource`。 |
| `ImageRadioButton` | 图片单选按钮。支持 `IsCheckedBinding` 和选中/未选中两套图片资源。 |
| `TabControl` | 标签页容器。`SelectedIndexBinding` 绑定当前选中标签索引。 |
| `HtmlPanel` | H5/HTML 内容显示面板。`TextBinding` 绑定 HTML 内容字段。 |
| `ScrollViewer` | 滚动容器。`PanningMode` 设置滑动方向。 |
| `ItemsControl` | 列表容器。`ItemsSourceBinding` 绑定集合数据，`ItemTemplate` 定义每一项模板。 |
| `Border` | 边框容器，常作为布局容器包裹子控件。 |
| `Viewbox` | 缩放容器，可让内部内容按比例缩放。 |

##### WallPage.xml 参数说明

| 参数/节点 | 说明 |
|---|---|
| `Width` / `Height` | 页面设计分辨率。当前配置为 `3840×2160`，表示按 4K 横屏设计。 |
| `Top` / `Left` | 页面在屏幕坐标系中的起始位置，通常保持 `0`。 |
| `OffsetY` | 商品瀑布流整体向下偏移的像素值，用于给顶部标题栏、分类栏等留出空间。 |
| `Rows/Row@Speed` | 瀑布流每一行的横向滚动速度。负值表示向左滚动，数值越大滚动越快。 |
| `Layout` | 布局类型，见下方详细说明。可选 `WaterFlow`（瀑布流）或 `FiveGrid`（五宫格）。 |

**布局类型说明**

- **`WaterFlow`（瀑布流）**：商品按原始图片比例横向排列，每行高度固定，系统每次把新商品放到“最短行”。整屏商品持续从右向左滚动，适合展示大量不同尺寸的商品图。
- **`FiveGrid`（五宫格）**：页面由“大图单元格”和“四宫格单元格”交替组成。每个大图单元格占一个固定大高度；四宫格单元格内部再细分为 2×2 共 4 张小图。整体仍以行横向滚动，但视觉节奏更适合突出部分重点商品。

**单元格配置节点**

| 节点 | 适用场景 | 说明 |
|---|---|---|
| `WallItem` | 普通商品 | 配置普通商品卡片（背景图、商品图、名称、价格、年龄段等）。 |
| `BigWallItem` | 大商品 / 突出商品 | 配置大卡片样式，在 `WaterFlow` 中用于 `IsBig` 商品，在 `FiveGrid` 中用于大图单元格。 |
| `WallWaterItem` | 水滴详情浮层 | 用户点击/触摸商品后，商品被“水滴”放大吸附，进入激活态，此时使用该配置。包含大图轮播、喜欢按钮、详情按钮、价格、二维码等。 |

每个单元格内部通用子节点含义：

| 子节点 | 说明 |
|---|---|
| `Background` | 单元格背景图。`Enable` 控制是否显示；`UriKind` + `FileName` 指定图片路径。 |
| `ImageRect` | 商品主图显示区域。`Left/Top/Width/Height` 为相对单元格左上角的坐标；`Enable` 控制是否显示。 |
| `LikeRect` | 喜欢/收藏按钮区域。在水滴态（`WallWaterItem`）中通常用于触发收藏。 |
| `InfoRect` | 详情按钮区域。在水滴态中点击可打开商品详情页。 |
| `NameText` | 商品名称文本。支持 `FontSize`、`FontFamily`、`Foreground`、`MaxWidth`、`Enable`、`FontWeight`。 |
| `PriceText` | 商品价格文本。参数同 `NameText`。 |
| `PromPriceText` | 促销价格文本（划线价场景），仅在 `WallWaterItem` 等水滴态配置中使用。 |
| `AgeScopeText` | 年龄段文本，例如 `3-6岁`。 |
| `QrcodeRect` | 二维码显示区域（旧版水滴态使用，新版已移除）。 |

**固定 UI 控件（`Canvas/Controls`）**

| 节点 | 说明 |
|---|---|
| `ImageButton` | 图片按钮，可配置 `FileSource` 图片路径和 `Event` 触发事件（如 `ToggleSearchEvent` 切换搜索面板）。 |
| `SearchPanel` | 搜索面板，`IsShow` 控制初始是否显示。 |
| `SearchResultPanel` | 搜索结果列表面板。 |
| `CustomElement` | 自定义 WPF 控件，通过 `ActivatorInfo` 指定程序集和类型名。 |

**其他配置节点**

| 节点 | 说明 |
|---|---|
| `TimeoutIndicator` | 空闲超时动画。`Timeout` 为超时秒数，无人操作后显示等待/屏保动画。 |
| `Prices/Price` | 搜索面板中的价格区间筛选列表，如 `0-999`、`1000-1999`。 |
| `Keywords` | 搜索面板中的关键字列表。 |
| `Filters/CategoryProductFilter` | 商品过滤器。`Enable` 控制是否启用，`CategoryIds` 中指定允许显示的分类 ID。 |
| `IsShowMainColors` | 是否启用按颜色搜索。 |
| `IsShowMainCategorys` | 是否启用按分类搜索。 |
| `IsShowKeywords` | 是否启用关键字搜索。 |
| `IsMainTags` | 是否启用标签搜索。 |
| `ShowSetting` | 瀑布流中是否显示分类入口、红包入口、搜索分类入口。属性：`IsShowCategory`、`IsShowCoupon`、`IsShowSearchCategory`。 |
| `GuessLike/Enable` | 是否启用“猜你喜欢”推荐。 |
| `GuessLike/Count` | “猜你喜欢”推荐商品数量。 |

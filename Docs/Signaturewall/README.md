# 签字墙

屏幕上进行手写签名时，签名内容可以实时飞行展示在屏幕上。签名飞屏控件通常具有美观、壮观的外观效果，能够吸引用户的眼球，提高用户的体验感受。一些高级的签名飞屏控件还可以实现特效效果。

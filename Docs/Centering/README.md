# 轴对中以及轴间距检测介绍

非接触式激光对中设备，对中设备为瑞典产品，测距设备为日本产品，均为行业头部，精准稳定 厂家出品蓝牙连接 精度高 可精确到 0.01mm 专用工业 PAD（防摔）便携性高，方便操作 使用原生平台开发，使用 wifi 传输字段信息而非传图有后台系统可以记录测量值，测量时间，操作人 ID，方便回溯，保留接口对接能力，可对接公司 MES 等系统，可延展性高。
